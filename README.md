# 🤖 Bluetooth Controlled Robot Car

This project is a Bluetooth-controlled robot car built using Arduino for real-time wireless control and motor-based navigation.

## 📌 Overview

The robot receives commands and moves accordingly using motor control logic implemented in Arduino.

## ⚙️ Components Used

* Arduino (Uno or similar)
* Bluetooth module (HC-05/HC-06)
* Motor driver (L293D / Adafruit Motor Shield)
* DC motors
* Power supply

## 🔧 Working

* The device sends commands via Bluetooth.
* Arduino reads the commands using serial communication.
* Based on the command:

  * F → Forward
  * B → Backward
  * L → Left
  * R → Right

## 📦 Requirements

* Arduino IDE
* Adafruit Motor Shield Library (AFMotor)
